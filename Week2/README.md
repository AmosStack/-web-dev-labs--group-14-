# Week 2 Web Lab

This folder contains the Week 2 pages for Group 14. The pages demonstrate basic multi-page HTML structure, use of local assets, and organisation of content for a small website.

## Weekly Goals

- Practice creating multi-page navigation and consistent layout.
- Present a simple timetable for class/work sessions.
- Use local screenshots and images to illustrate content.
- Keep pages accessible and linkable without server-side code.

## Project Structure

- [Week2/index.html](Week2/index.html) - Home page with navigation and overview.
- [Week2/contact.html](Week2/contact.html) - Contact information and group details.
- [Week2/timetable.html](Week2/timetable.html) - Timetable page showing schedules.
- [Week2/Screenshots/](Week2/Screenshots/) - Folder with screenshot image assets.

## Page Summaries

- **Home (`index.html`)**: Introduction to the week's work, navigation to other pages, and a local image or screenshot illustrating the site.
- **Contact (`contact.html`)**: Group contact details and roles; static contact info for project members.
- **Timetable (`timetable.html`)**: A simple table or list showing the weekly schedule or important dates.

## How to Open

1. Open the `Week2` folder in your browser or editor.
2. Open [Week2/index.html](Week2/index.html) to start from the home page.
3. Use the navigation links to move between pages.

## Notes

- Pages are plain HTML and use relative paths for local assets.
- Screenshots in `Screenshots/` illustrate page layouts — keep filenames intact so links work.
