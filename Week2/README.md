# Week 2 Web Lab

This folder contains the Week 2 pages for the group web lab.

## Project Structure

- `Week2/index.html` - Home page
- `Week2/contact.html` - Contact page
- `Week2/timetable.html` - Timetable page
- `Week2/Screenshots/` - Screenshot assets

## What the Site Includes

- Navigation links between the main Week 2 pages
- Basic HTML lists, headings, and paragraphs
- Local screenshots stored in the `Screenshots` folder

## How to Open

1. Open the `Week2` folder in your browser or editor.
2. Open `index.html` to start from the home page.
3. Use the navigation links to move between pages.

## Notes

- The pages use only HTML and local image assets.
- External links (if present) open in a new tab.
