# Week 1 Web Lab

This workspace contains a simple multi-page HTML website for Week 1.

## Project Structure

- `week1/index.html` - Home page
- `week1/about.html` - About page
- `week1/contact.html` - Contact page
- `week1/links.html` - External links page
- `week1/images/local-image.jpeg` - Local image used on the home page

## What the Site Includes

- Navigation links between all main pages
- A local image displayed from the `images` folder
- An external image on the links page
- Basic HTML lists, headings, and paragraphs
- Contact information and useful resource links

## How to Open

1. Open the `week1` folder in your browser or editor.
2. Open `index.html` to start from the home page.
3. Use the navigation links to move between pages.

## Notes

- The project uses only HTML and image assets.
- External links open in a new tab.
- The local image path is relative, so it works when the folder structure stays the same.
