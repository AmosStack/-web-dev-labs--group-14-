# Week 3 Web Lab

This folder contains the Week 3 page for Group 14. It demonstrates a simple personal webpage built with HTML and CSS, including internal navigation, a profile image, and a linked stylesheet.

## Weekly Goals

- Practice building a single-page layout with sections.
- Use both external and internal CSS for styling.
- Display a profile image and basic personal information.
- Organize content with semantic HTML elements.

## Project Structure

- [Week3/webpage.html](Week3/webpage.html) - Main personal webpage with About, Skills, and Daily Routine sections.
- [Week3/web.css](Week3/web.css) - External stylesheet used by the page.
- [Week3/am.jpeg](Week3/am.jpeg) - Profile image displayed in the About section.

## Page Summary

- **Webpage (`webpage.html`)**: A personal profile page for Amos with navigation links to page sections, a short bio, listed skills, and a daily routine.

## How to Open

1. Open the `Week3` folder in your editor.
2. Open [Week3/webpage.html](Week3/webpage.html) in a browser.
3. Use the navigation links at the top of the page to jump between sections.

## Notes

- The page uses a linked stylesheet and relative image paths, so keep the files in the same folder.
- The navigation links point to section IDs on the same page.
